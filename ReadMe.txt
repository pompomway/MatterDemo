本Demoはnode.jsで実行できます。

node.jsのバージョンは7.3.0以上推奨。

IDEは必要がありません。コマンドプロンプトで起動できます。


Step1. コマンドプロンプトでMatterTagフォルダに移動する。ex: C:\...\Desktop\MatterTag>

Step2. コマンドプロンプトに　npm init -y　を入力する。

Step3. コマンドプロンプトに　npm install express　を入力する。

Step4. コマンドプロンプトに　npm install path　を入力する。

Step5. コマンドプロンプトに　node server.js　を入力する。

Step6. ブラウザで　http://localhost:3000/crud2　にアクセス。
